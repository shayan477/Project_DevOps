What is DevOps?

DevOps is a set of practices that combines software development (Dev) and IT operations (Ops). It aims to shorten the development lifecycle and provide continuous delivery with high software quality.

How This Project Works

This project showcases a fully automated DevOps pipeline:

  • Terraform provisions an Ubuntu VM on Azure.
  • Ansible installs and configures Apache to serve this web page.
  • Jenkins (optionally) ties it all together in a CI/CD pipeline.
